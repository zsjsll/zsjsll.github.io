# zsjsll.github.io
## 下载说明
先把这个项目克隆下来，然后执行：
```
npm install hexo
npm install
npm hexo-deployer-git --save  #git 提交
npm install hexo-generator-searchdb --save   # 搜索服务
npm install https://github.com/CodeFalling/hexo-asset-image --save #插入图片
npm install hexo-asset-image --save #同上 选一个即可
```
